Adds social share buttons to posts without loading remote javascript libraries.

== Description ==

Social share buttons curently supported are:
* Facebook like;
* Twitter tweet.

Settigns page allows to quickly customize Twitter username and Facebook App ID. Twitter username setting can be left blank, then tweets will not contain via @username. Rest of the settings used in Facebook like , like, site url, site name are retrived from WordPress settings.
Plugin will attempt to get post first attachment image thumbnail to use for Facebook.

== Installation ==

1. Upload `simple-social` plugin directory to the `/wp-content/plugins/` directory
2. Add Facebook App ID and Twitter name in Plugins > Simple Social

== Frequently Asked Questions ==

= Why aren't more buttons supported? =

I'm working on adding more sharing buttons for next version.

= How can I change button background images? =

Currently there's no clean way to that. Option to change images is planned for next release.
